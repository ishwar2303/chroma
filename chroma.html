Chroma is a syntax highlighting library written in javascript.

It is lightweight and easy to use.

How to use it?

1. 
Place chroma folder in the root directory of project.
2. 
Include some markup for code to be highlighted.
<div chroma="true" language="html"> Place your code here... </div>
3. 
Include Chroma js in the end of body.
<script src="chroma/dist/chroma.js"></script>
How does it work?
Chroma analyze your code based on language you choose, It process regular expression on code and wrap matches in <span></span> tag.
Corresponding css classes are attached and code is ready to serve.
How to use it in your script
/* * @param string, code to be highlighted * @param string, language * returns highlighted code */ var highlighted = Chroma.pretty(code, language) /* * Apply theme */ var options = { theme : 'rosy' } Chroma.setOptions(options)

Understanding Chroma attributes

Required

chroma="true" Chroma need this attribute to target your code
language="css" Specifies language to apply related regular expressions
If you want to highlight sql than you should change language="css" to language="sql"

Optional

linepad="true" Shows line number
preloader="true" Loading animation before revealing code
header="true" Header attached to code with heading and copy button
copy="true" Set Copy button
heading="This is a heading" Shows the value in header
By default all options are applied to remove any option set value to "false"
