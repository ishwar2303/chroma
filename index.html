Chroma

Let's make your code distinct and beautiful.

What is this?

Chroma is a code syntax highlighting library written in Javascript.

It is completely themable via CSS.

How it works?

Chroma is fully automated works on its own

Picks your code analyze it based on data-lang="" option

Apply regular expression and embed matches in span tag.

Corresponding css classes are attached

Installation

Download chroma-code-assets folder from github.

Include folder in your root directory of project.

How to use?

HTML

Place your code in textarea

HTML

Include script in the end of body

Let script handle everything for you.
