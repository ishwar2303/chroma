Let's make your code beautiful

Chroma is a syntax highlighting library written in javascript.

It is lightweight and easy to use.

Theme
Bold
Capital

How to use it?

1. 
Place chroma folder in the root directory of project.
2. 
Include some markup for code to be highlighted.
<div chroma="true" language="html"> Place your code here... </div>
3. 
Include Chroma js in the end of body.
<script src="chroma/dist/chroma.js"></script>
4. 
Finally include Chroma css in head
<link rel="stylesheet" href="chroma/css/chroma.css"> <link rel="stylesheet" href="chroma/themes/dark.css">
How does it work?
Chroma analyze your code based on language you choose, It process regular expression on code and wrap matches in <span></span> tag.
Corresponding css classes are attached and code is ready to serve.
How to use it in your script?
/* * @param string, code to be highlighted * @param string, language * returns highlighted code */ var highlighted = Chroma.pretty(code, language) /* * Apply theme */ var options = { theme : 'dark', bold : false, // if true keywords will appear bold capital : false // if true keywords will be capital } Chroma.setOptions(options)
pretty(code, language) method takes two parameter
1. code to be highlighted
2. language for which code needs to be highlighted

Understanding Chroma attributes

Required

  • chroma="true" Chroma need this attribute to target your code
  • language="css" Specifies language to apply related regular expressions
  • If you want to highlight sql than you should change language="css" to language="sql"

Optional

  • Use linepad="true" to Show line number
  • For loading animation set preloader="3" value is in number of seconds
  • To set header use header="true"
  • Set copy button to header copy="true"
  • If code is very large use height="300px" doing so will set the max height of code block to 300px and scroll is set
    Height value can be according to your preference don't forget to write px with number
  • To give heading in the header use heading="This is a heading"
  • By default all options are applied to remove any option set value to "false"
